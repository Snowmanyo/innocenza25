# 🛍️ 代購團管理系統

> 專為代購團隊設計的訂單管理工具，支援多人即時同步、順位管理、小卡計算等功能

---

## ✨ 功能特色

### 📱 核心功能
- ✅ **多人即時同步** - 資料儲存在 Google Sheets，團隊成員同步查看
- ✅ **訂單管理** - 新增、編輯、刪除客人訂單
- ✅ **順位系統** - 自動計算每個商品的購買順位
- ✅ **小卡計算** - 根據韓元消費自動計算獲得小卡數（每 ₩50,000 = 1 張）
- ✅ **已購標記** - 勾選已購買品項，即時更新金額統計
- ✅ **總覽表** - 一鍵查看所有客人的訂購彙整
- ✅ **密碼保護** - 保護客人隱私資料

### 🎨 介面設計
- 📱 響應式設計 - 支援手機、平板、電腦
- 🎴 卡片式訂單 - 左右滑動查看，清晰直覺
- 🔢 自動編號 - 商品自動編號（如 1-1, 1-2），方便管理
- 💰 雙幣顯示 - 同時顯示韓元和台幣價格

## 🚀 快速開始

### 步驟 1: 建立 Google Sheets
1. 到 [Google Sheets](https://sheets.google.com) 建立新試算表
2. 複製試算表 ID（網址中的一長串）

### 步驟 2: 設定 Google Apps Script
1. 到 [Google Apps Script](https://script.google.com)
2. 建立新專案，貼上 `apps-script-final.js` 的程式碼
3. 修改第 2 行填入你的試算表 ID
4. 執行授權並部署為「網頁應用程式」
5. 複製部署後的網址

### 步驟 3: 設定 HTML
1. 打開 `index.html`
2. 第 851 行：貼上 Apps Script 網址
3. 第 1745 行：修改登入密碼（重要！）

### 步驟 4: 部署到 GitHub Pages
1. 建立 GitHub repository
2. 上傳 HTML（檔名必須是 `index.html`）
3. 啟用 GitHub Pages
4. 完成！

詳細步驟請見下方「安裝教學」。

## 📖 使用說明

### 登入系統
- 打開網址，輸入密碼
- 登入後 24 小時內免重複輸入

### 新增訂單
1. 點「➕ 新增訂單」
2. 填寫客人資料
3. 選擇商品和款式
4. 點「加入品項」（每次 1 個）
5. 點「確認儲存」

### 標記已購買
- 勾選「已購買」即可
- 自動計算金額和小卡數

### 查看總覽
- 點選「總覽表」頁籤
- 或直接在 Google Sheets 查看

## 🔧 客製化設定

### 修改商品資料
找到 `products` 陣列（第 854 行）：
```javascript
{ id: 1, name: "商品名稱", krw: 韓元, price: 台幣, variants: ["款式"] }
```

### 修改密碼
找到第 1745 行：
```javascript
const CORRECT_PASSWORD = '你的密碼';
```

### 修改小卡門檻
找到 `calculateGiftCards` 函數：
```javascript
return Math.floor(totalKrw / 50000); // 改成你要的門檻
```

## 🐛 疑難排解

### 無法載入資料
- 清除瀏覽器快取（Ctrl + Shift + R）
- 確認 Apps Script 網址正確
- 檢查網路連線

### 資料沒有同步
- 重新整理頁面（F5）
- 確認都用同一個網址

### 電腦無法使用
- 使用無痕模式測試
- 清除網站快取（F12 → Application → Clear site data）
